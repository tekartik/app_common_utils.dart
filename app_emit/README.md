# app_emit

Emit controller (Stream, FutureOr, Future) providing stream/future like

## Setup

In your `pubspec.yaml`:

```yaml
dependencies:
  tekartik_app_emit:
    git:
      url: https://github.com/tekartik/app_common_utils.dart
      ref: dart2_3
      path: app_emit
    version: '>=0.1.2'
```
