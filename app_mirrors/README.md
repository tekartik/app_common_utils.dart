# Mirrors support

Code Generator running on the VM. Allows importing safely `mirrors.dart` in any project.

## Setup

In your `pubspec.yaml`:

```yaml
dependencies:
  tekartik_app_mirrors:
    git:
      url: https://github.com/tekartik/app_common_utils.dart
      path: app_mirrors
    version: '>=0.1.0'
```
